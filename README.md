# 大模型系统提示泄漏(System Prompts Leaks)

<img src="./img/logo.svg" alt="Logo" style="max-width: 100%; height: auto; width: 300px;">

主要收集各类大型语言模型(LLMs)的系统提示(System Prompts)示例，帮助开发者更好地理解和使用这些模型。

## 包含模型

- ChatGPT
- Claude 
- Qwen (通义千问)
- Gemini
- 更多模型持续添加中...
